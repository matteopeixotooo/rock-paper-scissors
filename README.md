# Pedra, Papel e Tesoura em Python

Jogo clássico Pedra, Papel e Tesoura jogado no terminal com arte ASCII.

## Como jogar

Digite:
0 = Pedra  
1 = Papel  
2 = Tesoura  

O computador escolhe aleatoriamente e o resultado é exibido.

## Requisitos

- Python 3.x

## Executar

```bash
git clone https://github.com/seu-usuario/pedra-papel-tesoura.git
cd pedra-papel-tesoura
python main.py
